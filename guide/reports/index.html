Reports ​

Reports let you generate documents from your project data — Bills of Materials, production spreadsheets, QR stickers, quality control PDFs, and more.

  • Report Status

Creating a Report ​

  1. Open a project and go to the Reports tab, or go to Company Reports for company-wide reports.
  2. Click the New Report button.
  3. In the report form dialog:
    • Select the document type (e.g., Plates BoM, Production Excel).
    • Choose which assemblies or modules to include.
    • Configure any options specific to the report type.
  4. Click Submit to queue the report for generation.

TIP

You can also click Save Without Generating to create the report entry without starting generation. This is useful if you want to configure the report now and generate it later.

Report Status ​

Once submitted, a report goes through these stages:

StatusMeaning
WaitingQueued for processing
WorkingBeing generated (progress percentage shown)
ReadySuccessfully generated — click to download
FailedAn error occurred during generation

When a report fails, you'll receive an email notification.

Versioning ​

Each time you regenerate a report, a new version is created. All previous versions are kept and can be downloaded from the version history.

Design Updates

If a design's IFC file is updated after a report was generated, a warning icon will appear next to the report. This means the report may contain outdated data. You can regenerate it to include the latest design changes.

Content Selection ​

Most reports require you to select which content to include:

  • Assemblies / Modules — Select specific items from the project tree.
  • Part Batches — Some reports allow selecting by part batch instead, which groups parts across assemblies.

Project Reports vs. Company Reports ​

Project reports are generated from a specific project's data. Most report types are project-scoped, including all Bills of Materials and machine files.

Company reports are available when you need data across all projects. These include:

  • Production Excel
  • Worktime Excel
  • Quality Control Report
  • Issues Report

Company reports use a date range filter instead of content selection, and can optionally be filtered by production line.

Report Types ​

Bills of Materials ​

PDF documents listing the parts needed for your constructibles:

  • Plates BoM — All plate parts with quantities and dimensions.
  • Wood BoM — All wood/profile parts.
  • Per-Design BoM — One BoM per unique design.
  • Accessories BoM — Accessories and hardware list.
  • Custom Woods CSV — Wood parts exported as CSV.
  • Custom Plates CSV — Plate parts exported as CSV.
  • Custom Accessories CSV — Accessories exported as CSV.

Project Reports ​

  • Production Excel — Production planning spreadsheet.
  • Worktime Excel — Worker time tracking spreadsheet.
  • Quality Control Report — PDF summarizing quality control records.
  • Issues Report — PDF listing all issues found during production.
  • QR Stickers — Printable QR code labels for physical identification.

Machine Files ​

Machine-specific file formats for CNC equipment. See the Machine Files section for details.
