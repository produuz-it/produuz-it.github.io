# CHANGELOG

# 2022-03-10

  • Improvements in btl, mob2 and Plates BoM report.
  • Improved speed on report generation when having multiple times the same design.
  • Updated Android to 2022.1.

# 2022-02-24

  • Fixed bug in projects menu.
  • Fixed bug in left menu that got stucked production users in the managers mode.
  • Fixed bugs in mob2 generator.
  • Turned back to old details&docs system.
  • Added theme menu.
  • New specs setup based on docs uploads. New specs touch&release button in viewer.

# 2022-02-18

  • Updated PDF scanning operation. Now you can scan for DOC NAMES both into drawings and details.
  • Fixed visual problems in dark interface.
  • New 2touch detail&doc buttons in tablet mode viewer.
  • Improved walls area calculation.

# 2022-02-15

  • Production overview: Multirow arrangement in overlapping batches.
  • Production overview: fixed wrong overlapping batches.
  • Dynamic forms translated and added sections in forms.
  • ardis: New setting to setup the max number of rows in a file.
  • ardis: New setting to create a XLS file instead of a CSV file.
  • dstv: Improved performance and reliability.
  • mob2: RoboBrick stations creates shift by their own blocker settings.
  • mob2: Improved performance and reliability.
  • btl: Improved performance and reliability.

# 2022-02-11

  • Fixed some translations (some texts were hidden).
  • Production overview: see and set workers in production lines.
  • Production overview: Fixed forecasted hours widget.

# 2022-02-08

  • mob2: New frame matrix system. Now walls can mix designed nails with auto-generated nails for columns without nails in the draw.
  • mob2: Milling polygons now keeps the milling tool down.
  • IFC PS can be now downloaded or uploaded as .json files.
  • Machine setup can be now downloaded or uploaded as .json files.
  • Project documentation can be added now as a PDF or as a raw text.
  • Mouse scrolling improvements in PDF and IFC view.

# 2022-01-14

  • Production Overview (beta2): loose batch picker, autoscroll to now, custom batch expected time, sticky left column, week startdate fix, fixed bugs.
  • Project Overview.
  • Improved status update on URL direct reload.
  • Fixed hidden phases dialog.
  • JJS Files (beta): Now, the users can upload JJS files in produuz.it as an alternative from IFC files.
  • mob2: Added new feature for lockout areas.
  • mob2: Improved performance and reliability.
  • btl: Improved performance and reliability.

# 2022-01-10

  • Improved reliability in panel count matrix.
  • Fixed a bug in report settings. Some settins were not correctly stored on clone/edit.
  • Fixed RFID button in viewer.
  • Company Production Overview (beta).
  • Full plates selector in plates reports.
  • Android: First release.
  • mob2: Added support for triangular panels.
  • mob2: Auto-remove duplicate milling operations.
  • mob2: Improved performance and reliability.
  • btl: Improved performance and reliability.

# 2021-12-17

  • ardis: Fixed an issue for Ardis csv generation.
  • ardis: Swith Width and Lenth for Ardis csv export.
  • mob2: Shift feature now based en machine Zeros.
  • mob2: Inside height is count now from the lowest level in frame parts.
  • mob2: Added a new feature for layers generation. Now the user can select between 3 options for generate the layers that corresponds to a certain machine, if the operations coordinates are above machine sizes:
  • Generate anyway: ignore the machine sizes, and generate the operations.
  • Do not generate the layer: if any operations is above machine sizes, the layer will not be generated.
  • Trim operations up to max size: the operations that exceeds machine sizes, will be trimmed up to the machine limit.
  • mob2: Added a new feature for twin nailings patterns. Now the user can select between the regular pattern, or a new one optimized for large panels.

# 2021-12-14

  • Improved operations status indications and disabled actions.
  • Solved several bugs in operations (generate_designs, detect_details, prepare_viewer).
  • Operation detect_details now has checked boxes by default since most users forget to check it.
  • IFC PS clone feature.
  • Change Stations menú upgrade.
  • mob2+mob1 files: Glueing and bricks bugs fixed.

# 2021-12-08

  • New file versioning in uploads. This solves some updating bugs.
  • New file status indicators (see: https://docs.produuz.it/operations.html#table-of-uploaded-files)
  • mob2+mob1 files: Spiral plate nailing for single nailing.
  • mob2+mob1 files: Frame nailing matrix fix for proper studs selection.
