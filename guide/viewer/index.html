3D Viewer ​

The Viewer is the main production interface in produuz.it. It displays 3D models and technical drawings of your constructibles (assemblies and modules), and provides tools for managing production checklists, issues, logistics, and more.

Accessing the Viewer ​

You can open the Viewer from the constructibles list in a project. There are two viewing modes:

  • Actual Viewer — For constructibles in production. Includes checklists, issues, RFID, and logistics features.
  • Design Viewer — For reviewing designs before production. Shows 3D models and drawings in read-only mode (no production tools).

Layout ​

The Viewer uses a three-panel layout:

PanelContent
Left3D model or technical drawing (PDF)
CenterVertical toolbar with action buttons
RightPart details (BOM) or specification documents

You can switch between three view modes using the buttons in the top-right corner:

  • Drawing mode — Left panel + toolbar only (focus on 3D/drawings).
  • Full mode — All three panels visible.
  • Detail mode — Toolbar + right panel only (focus on part details).

3D Model ​

Controls ​

  • Rotate — Click and drag to rotate the model.
  • Zoom — Scroll wheel or double-click to zoom in.
  • Pan — Right-click and drag to move around.
  • Recenter — Click the crosshair button to reset the view.
  • Front/Back — Toggle buttons to flip the camera perspective.

Selecting Parts ​

Click any part in the 3D model to select it. The right panel will show its properties, including:

  • Zone and reference number
  • Material type and dimensions
  • IFC properties and property sets
  • Creation date and revision number

Zone Selection ​

Use the zone toggle buttons to show or hide specific zones of the constructible. This is useful for focusing on a specific area during production.

For modules, you can also filter by sub-assembly using the assembly dropdown.

Advanced Tools ​

  • Measurement — Click the ruler icon to measure distances between parts in the 3D view.
  • Clipping Plane — Click the section icon to enable a cutting plane that reveals the interior structure.

Technical Drawings ​

The left panel can switch between the 3D model and technical drawings (PDFs). Use the buttons at the bottom of the left panel to select which drawing to display.

  • Drawings are filtered by your selected production station when applicable.
  • Multi-page drawings show navigation arrows to move between pages.
  • Zoom and pan controls work the same as in the 3D view.

Bill of Materials (BOM) ​

The right panel in Details mode shows the Bill of Materials for the current constructible.

Part List ​

A table listing all tracked parts with:

  • Reference number
  • Part category badge
  • Specifications
  • Zone location

Use the branch filter buttons (e.g., wood, metal, hardware) to show only parts from a specific construction branch.

Unknown Parts ​

A separate tab shows parts detected in the IFC file but not matched to the database. This helps identify parts that may need to be added to the system.

Part Drawings ​

When a part is selected, you can view its associated detail drawings and CAD files in the right panel.

Production Features ​

INFO

Production features are only available in the Actual Viewer (not in Design Viewer).

Checklist ​

The checklist button in the toolbar shows the production progress for the current constructible at the selected station.

  • Progress bar — Color-coded: red (not started), amber (in progress), green (completed).
  • Task list — Each task has a checkbox, and may require:
    • QR code scanning
    • Picture upload (opens camera interface)
    • Filling a custom form
    • Reviewing associated documents

Actions:

  • Start Production — Begin work on this station (required before checking tasks).
  • Pause — Temporarily stop work.
  • Station Job Completed — Mark the station as done (enabled when all tasks are checked).

Station Switching ​

Click the station button to change to a different production station. If previous stations haven't been completed, you'll be asked to confirm a force-start.

Issues ​

The issues button shows a badge with the count of active issues for this constructible.

  • View issues — See all issues with their title, description, category, status, and attached pictures.
  • Add issue — Create a new issue with a category, title, description, and optional picture upload. You can attach the current part information for context.

QR Code ​

Click the QR button to display a QR code for the current constructible. This can be printed and attached to the physical item for quick identification on the shop floor.

RFID Tag ​

Click the RFID button to register an RFID tag code for the current constructible. You can type the code manually or scan it using the camera.

Rack / Logistics ​

The rack button shows whether the constructible has been loaded into a delivery rack.

  • Load to Rack — Mark the item as placed in a rack.
  • Unload from Rack — Remove the rack assignment.
  • Shows a visual representation of the rack arrangement.

Specifications & Documents ​

Switch to the Docs mode in the right panel to view specification documents associated with the constructible. These may include finish specifications, hardware specs, or other technical documents in PDF or image format.
